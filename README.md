# 📝 MemoStack – Your Personal Space for Notes

MemoStack is a secure web-based notes management application built using the MERN stack. It allows users to create, view, edit, and delete notes with authentication and a clean user interface.

---

## 🚀 Features

- 🔐 User Authentication (JWT-based)
- 📝 Create, Read, Update, Delete (CRUD) Notes
- 🎨 Sticky Notes UI (Board View)
- 📂 Organized Notes List
- ⚡ Real-time Updates
- 🔒 Secure API with Token Authorization

---

## 🛠️ Tech Stack

### Frontend
- HTML
- CSS
- JavaScript (Vanilla JS)

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Mongoose)
